# TriAgnostics — Gut Health Assessment

A simulated Web3-style gut health assessment app built for the 8x hackathon.

**Live Demo:** https://triagnostics-8x-mobile-hackathon.vercel.app/

**Deck:** https://docs.google.com/presentation/d/1bzrfvN3IuRUz8Z7ncr0-md8XUIt6j-CT/edit?usp=sharing&ouid=112901307157703667136&rtpof=true&sd=true

## Overview

TriAgnostics is a data-driven SPA that guides users through a personalized gut-health assessment, simulating wallet connection, token rewards, premium plans, and bonus quiz flows.

## Features

- **Simulated Wallet Connection**: Fake MetaMask-style wallet flow (no real blockchain)
- **14-Question Quiz**: Grouped into three categories with choice/date/text inputs
- **Profile Analysis**: Assigns one of four profiles based on scoring algorithm
- **Fake Rewards System**: Earn demo tokens by connecting Apple Fitness
- **Premium Plans**: Unlock bonus features with simulated subscription
- **Bonus Quiz**: Extra questions for additional token rewards
- **Progress Tracking**: Charts showing simulated wellness trends

## Tech Stack

- React 18 + Vite 5
- TypeScript
- Tailwind CSS
- React Router 6
- Zustand (state management)
- recharts (data visualization)
- lucide-react (icons)
